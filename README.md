# Claude Code Workspace Template

A structured workspace template for working with Claude Code as a persistent agent 
assistant across sessions. Designed around a `/prime` command that loads your context 
at the start of every session — keeping Claude oriented without bloat.

## What's Included

- `CLAUDE.md` — Core workspace configuration, auto-loaded by Claude Code
- `.claude/commands/` — Slash commands: `/prime`, `/create-plan`, `/implement`
- `context/` — Your personal, role, and business context files (fill these in)
- `plans/` — Implementation plans generated by `/create-plan`
- `outputs/` — Deliverables and work products
- `reference/` — Reusable templates and patterns
- `scripts/` — Automation scripts
- `SKILL.md` + Python scripts — Tools for creating and packaging Claude Code skills

## How to Use

1. Clone or download this repo
2. Open in VS Code or Claude Code
3. Fill in the `context/` files with your role, business, and goals
4. Run `/prime` at the start of every Claude Code session
